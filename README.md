# nest-boilerplate

## Installation

```bash
npm install
```

## Running the app

### Dependencies

```bash
docker-compose up
```

### App

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Migrations

### Generate a migration

```bash
npm run migration:generate <MIGRATION_NAME>
```

### Create a migration

```bash
npm run migration:create <MIGRATION_NAME>
```

### Run migrations

```bash
npm run migration:run
```

## Test

```bash
# unit tests
npm run test

# unit tests with watch
npm run test:watch

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Swagger

Browse to `/api` to get to the Swagger UI
